# NJ Energy Study

## Run it

```
pip install flask
python app.py
```

Open http://localhost:5000

## Files

```
nj-energy-study/
├── app.py              # The app
├── requirements.txt    # flask
└── templates/
    └── index.html      # The website
```

Data gets saved to `data/` and `uploads/` (created automatically).

## EC2

```
pip3 install flask
python3 app.py
```

Open port 5000 in security group. Done.
